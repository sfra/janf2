MVC is a framework working on node javascript enviroment. It provides the standard methods known from MVC (model view controller) pattern. It is assumed to be similar to the frameworks known form Java (Java Server Pages) or PHP (Laravel). It is a different approach then in frameworks working on JavaScript.

Web page is activated as an instantion of action coitained in a controller.

It provides wide support of layouts filled programatically or by the values contained in config files (View).

Moreover, it provides full support of some databases (Mysql, PostgreSql).
